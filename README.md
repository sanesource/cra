# sanesource/cra

Minimal Create-React-App

## Installation

Download and install the script

```
curl -s https://raw.githubusercontent.com/sanesource/cra/main/cra > /usr/local/bin/cra && chmod 777 /usr/local/bin/cra 
```

## Usage

```sh
# By path
cra path/to/app/

# By name
cra hello-world-app

# In current directory
cra .
```
